## 概要

Sovereignは、アメリカの公共信用創設をテーマにした**ハミルトン・システム・モノポリー風ボードゲーム**であり、さらに**ソロ／デジタル版**としても提供されており、同じルールをブラウザ上でローカルに実行し、2つの決定論的なスクリプト化された対戦相手と対戦できます。

- **ボードゲーム** — 印刷可能な34ページの版。40マス、22の物件＋4つのルート＋2つの機関、8つの色分けされたシステム、固定された歴史的順序で配置された7つの連邦議会法、4つのプレイヤーロール、3つの共有トラック（公共信用・公共抵抗・工業能力）、12+12のイベントカード。財務省以外に、商人（Merchant）と製造業者（Manufacturer）という2つの実行可能な経済的戦略。v0.2のバランス調整は完了し、変更はありません。
- **デジタルモード** — 単一の独立したHTMLファイル。回路ベースのゲーム終了条件：いずれかのプレイヤーが財務省を4回通過するとゲームは終了します。平均ゲーム時間は**約22ラウンド（約66ターン）**。最終会計時に、最も影響力の高いプレイヤーが勝利します。必ずしも最初にボードを一周したプレイヤーとは限りません。対戦相手の存在を示すレイヤー：可視の影響力ランキング＋対戦相手ごとの姿勢を示す線。戦略的な深さのレイヤー：3つのプロファイルロックされた特殊アクション、タイミングウィンドウを持つ6つのHANDカード、改革回復アクション、多段階の信用スパイラル（公共の疑念→危機→パニック→デフォルト）と、負債の返済、予測、そして改革による救済策。戦略的な展開のレイヤー：8ラウンドから各ラウンドで発動する8つの連邦時代イベント、3つのプロファイルビジョンとゲーム終了時のボーナス。記録者のナレーションレイヤー：14のイベントに関連する歴史的なバナーと、有益なTier B（詳細情報のポップオーバー、記録者の記録百科事典、用語集ツールチップ）、連邦主義者の論文と創設者オンラインからの実際の引用、×ボタンで閉じることができる永続的なメッセージ。視覚効果とサウンド：数値のアニメーション、ZzFXによる手続き型オーディオ（13種類のキュー）、映画のような／通常／高速の速度設定。決定論的なmulberry32 RNG、スクリプト化されたAI対戦相手、ハッシュ整合性による保存／読み込み、リプレイスクラバー、デザイナーが制御するバッチシミュレーションツール。
- **3つの現実的な勝利への道** — 財務省、商人、製造業者のいずれも、それぞれ意味のある勝利を収めることができます。財務省が最も強力であり、歴史と一致しています。公共信用と連邦金融は、ハミルトンの主要な経済的手段でした。各プロファイルは異なってプレイでき、独自の特殊アクションと、目指すべきプロファイルビジョンがあります。

## クイックスタート

### ブラウザでプレイ（インストール不要）

```bash
npx @mcptoolshop/sovereign
```

CLI（コマンドラインインターフェース）を開くと、ゲームがデフォルトのブラウザで起動します。インストーラー、サーバー、インターネット接続は必要ありません。

その他のモード：

```bash
npx @mcptoolshop/sovereign --print    # Open the printable 34-sheet board game
npx @mcptoolshop/sovereign --start    # Open the audience-routed landing page
npx @mcptoolshop/sovereign --path     # Print the playable HTML file path
npx @mcptoolshop/sovereign --help     # All flags
```

### オンラインでプレイ

ホストされているランディングページ（**<https://mcp-tool-shop-org.github.io/sovereign/>**）を開き、デジタルゲームをクリックしてプレイします。

### 印刷してプレイ

印刷可能なボードゲームは、単一の34ページのHTMLドキュメントです。パッケージから`release/board-game/sovereign-board-game.html`を開くか、ダウンロードして、`Cmd/Ctrl-P → PDFとして保存 → US Letter → 100%の縮尺`を選択します。切り取ってプレイします。

### オフラインリリースバンドル

各タグ付きリリースには、`sovereign-vX.Y.Z-release.zip`という名前のバンドルが、GitHubリリースページに添付されています。ダウンロードして解凍し、`00-START-HERE.html`を開くと、対象ユーザーに合わせたエントリーポイントが表示されます。すべてオフラインで実行できます。

## このゲームが存在する理由

Sovereignのテーマは、**公共信用＋連邦金融**がアレクサンダー・ハミルトンの主要な経済的手段であったということです。しかし、ハミルトン・システム・ゲームでは、**商業**と**産業**も実行可能な勝利への道として存在する必要があります。バランス調整（v0.2 → v0.10）は、9つのバージョンにわたる、証拠に基づいた取り組みであり、財務省を最も強力なプロファイルとして維持し（歴史と一致）、デザインを単一の戦略ゲームに陥らせないようにしました。

バージョンごとの進化については、[`CHANGELOG.md`](./CHANGELOG.md) を参照してください。

## 決定論

同じシード＋同じプレイヤーの決定＝実行ごとにバイト単位で同一の記録、ブラウザ、オペレーティングシステム。

- 単一のRNG：`mulberry32(state.rngSeed)`。
- 対戦相手の決定：可視の状態の純粋な関数であり、すべての決定とその決定を引き起こしたルールが記録に記録されます。
- 保存／読み込みの往復処理で、状態のハッシュが保持されます。
- リプレイは、`initialState(seed) + decisionLog`から再構築されます。
- v0.2 → v0.10のバランス調整中に、1,000以上の決定論的なゲームで検証されました。

## 脅威モデルとデータ処理

Sovereignは、スタンドアロンのブラウザベースのボードゲームです。CLIを開くと、ローカルのHTMLファイルがデフォルトのブラウザで開きます。サーバー、ネットワーク接続、アカウント、クラウド同期はありません。

- **アクセスされたデータ:** `release/` に含まれる HTML ファイル（読み取り専用）と、`sovereign.autosave` キーの下にある `localStorage`（ゲームの保存状態のみ）。
- **アクセスされないデータ:** パッケージディレクトリ外のファイルシステムへのアクセス、あらゆる種類のネットワークリクエスト、テレメトリ、分析、認証情報。
- **必要な権限:** OS のデフォルトブラウザを起動する機能、パッケージ自身のファイルを読む機能、ブラウザの `localStorage`（オプション）。
- **テレメトリは一切行いません。** シミュレーターの「テレメトリ」機能とは、ブラウザ内の台帳から派生したローカルのゲーム分析レポートを指します。これらのレポートは、決してあなたのデバイスから送信されません。

脆弱性に関する報告と、完全なセキュリティポリシーについては、[`SECURITY.md`](./SECURITY.md) を参照してください。

## 機能

- **ソロのサーキット勝利ゲーム:** 2 人のスクリプト化された対戦相手（デフォルトでは財務省/財務と商人/インフラストラクチャ、バッチプレイでは製造/産業）。いずれかのプレイヤーが財務省オープンの 4 回目の通過を完了するとゲームは終了します。平均ラウンド数約 22 ラウンド / 約 66 ターン。最終会計時の影響力が最も大きいプレイヤーが勝利します。
- **ライバルの存在（v1.5 レイヤー）:** 視覚的に表示される影響力のランキングと、各ライバルの動きを、レースにおける *あなたの* 立ち位置を基準として示す姿勢ライン。これにより、並行したソロゲームのような感覚がなくなり、対戦相手は対戦相手として認識されます。これはあくまで表示上の機能であり、ハッシュ化された台帳には書き込まれません。
- **クレジット・スパイラル（v1.5 レイヤー）:** 公的クレジットの失敗が、現在ではより明確に感じられ、累積し、回復可能になりました。クレジットが低い場合に、債務返済のための現金拠出が行われ、デフォルトへの加速が事前に示され、その傾向が予測され、改革措置が現実的な解決策として提示されます。これにより、連邦政府の公的クレジットが重要であった理由が直接的に伝わります。既存の失敗階層（国民の疑念 → 危機 → パニック → デフォルト）を包含し、保存/リプレイは完全に決定論的に行われます。
- **効果音とサウンド（v1.5 レイヤー）:** 数値のアニメーションに、増加/減少の非対称性、13 のキューにわたる ZzFX による手続き型のオーディオ、アクションの振り付け、および速度設定（シネマティック / 通常 / 高速インスタント）を追加。高速インスタントでは、すべてのアニメーションがスキップされ、高速プレイとアクセシビリティが向上します。キーボード操作、モーションの削減、スクリーンリーダーのサポートを全体で提供します。
- **戦略的な深み（v1.2 レイヤー）:** 3 つのプロファイルにロックされた特別なアクション（連邦債の発行 / ルート契約の仲介 / ワークショップの設立）、タイミングウィンドウを持つ 6 つの HAND カード（ハンド上限 2）、改革による回復アクション。
- **戦略的な展開（v1.3 レイヤー）:** 8 ラウンド以降の各ラウンドで発動する 8 つの連邦時代イベント（5 つの選択肢 + 3 つの自動）、3 つのプロファイルビジョン（連邦クレジットアーキテクト / コマース・ソブリン / 産業創始者）と、ゲーム終了時のボーナス。3 つのビジョンはすべて達成可能です。
- **記録者（v1.4 レイヤー）:** 名前付きの三人称の歴史的ナレーション。14 個のイベントに関連付けられたバナー（7 つのアクト / 連邦時代の開始 / 疑念 / 危機 / パニック / デフォルト / 反乱 / 改革 / ビジョン / 最終会計）。すべての引用は、founders.archives.gov、Wikisource、および米国議会図書館の資料と照合して検証されています。失敗したアクトは、現実の歴史に対する反事実として語られます（「私たちの歴史では、ハミルトンの資金調達法は 1790 年 7 月に 32 対 29 で可決されました。あなたの共和国では、兵士に対する差別が十分な票を集め、その道を阻止しました」）。永続的なフォイルボーダーのトーストが表示され、× で閉じることができます。ナレーションのオン/最小/オフ設定を尊重します。
- **記録者 Tier B — 情報レイヤー（v1.5）:** 主要なメカニズムに関する 15 個の「詳細はこちら」ポップオーバー、**記録者の台帳**百科事典（27 個の検証された歴史的引用に加えて、アクト、連邦時代イベント、クレジット階層、およびビジョンを 1 つの参照オーバーレイにまとめたもの）、および 10 個の用語集ツールチップ。この機能により、時代背景が、実際に閲覧可能な歴史レイヤーに変わります。
- **オンボーディング（v1.5 レイヤー）:** 初めてプレイするプレイヤーがゲームの基本的な流れを理解できるように、Swift-Start ガイド付きの「1790 年の資金調達に関する議論」のオープニングを提供します。さらに、すべての操作のコストと結果を、操作を実行する前に表示する、何も隠さないホバー/フォーカス表示機能も追加しました。
- **決定論的な AI:** すべての対戦相手の決定は、視覚的に確認できる状態の純粋な関数であり、台帳に記録された理由があります。LLM や不透明な魔法は使用していません。
- **8 つのゲーム画面:** ボード、財務省パネル、資産インスペクター、イベントドロワー、連邦議会の法案、共有トラック、ターンログ/台帳、ゲーム終了レポート。
- **オークション:** 拒否された資産は、プロファイルに基づいてスクリプト化された入札が行われるマルチプレイヤーオークションにかけられます。
- **保存/読み込み:** 各ターンの終わりに `localStorage` に自動保存、手動による JSON エクスポート/インポート、読み込み時のハッシュ整合性チェック、バージョンによる制限。
- **リプレイ:** 完了したゲームを完全にスクロールして再生できます。読み取り専用です。シードと決定ログから再構築され、緑色の整合性マークが表示されます。
- **バッチシミュレーション:** 任意のプロファイルトリプレットに対して、10 / 50 / 100 の決定論的なゲームを実行し、バランス分析のために JSON + HTML レポートをエクスポートします。
- **歴史的なナレーション:** 台帳から派生した 25 個のエントリのライブラリ（40 ～ 60 語のデフォルト、150 ～ 200 語の拡張、約 300 ～ 500 語のゲーム終了時の共和国の概要）。状態は決して変更されません。
- **アクセシビリティの最低基準:** 完全なキーボード操作、フォーカスインジケーター、スクリーンリーダーで意味のあるラベル、トラック値はテキストとしてだけでなく、マーカーとしても読み取れるようにする、14px 以上の最小フォントサイズ、モーションの削減への対応。

## プロファイルラインナップ（v0.10 バランスのベースライン）

| プロファイル | 資産の優先順位 | 強み | 弱み |
|--------------------------------|---------------------------------------------------------------|----------------------|-------------------------------------|
| **Treasury / Finance**         | NF > 州債 > 歳入債 > 銀行 > 造幣局 | 公的クレジットの増加 | インフラ収入なし |
| **Merchant / Infrastructure**  | ルート（すべて4つ）> コマース > 改善 > 歳入 | ルートラダー | 産業能力によるスコアリングなし |
| **Manufacturer / Industry**    | 製造 > 戦略的産業 > 改善 > 銀行 | 能力の倍増 | 序盤は遅いが、開始時にチャーターを取得 |

4 番目のコンセプトドキュメントのプロファイル（機会主義者 / キャッシュ）は延期されます。v0.10 のロックされた競争セットは 3 つです。

## ベータ版の注記

- **これは公開ベータ版です。** 改善され、楽しく遊べるように調整されていますが、まだ開発段階であり、時々不具合が見つかる可能性があります。不具合報告やフィードバックは、[問題追跡システム](https://github.com/mcp-tool-shop-org/sovereign/issues) で受け付けています。
- **最初にボードを制覇したからといって、勝つとは限りません。** ゲームは、プレイヤーがボードを4周した時点で終了しますが、最終的な影響力を最も多く持っているプレイヤーが勝者となります。経済的な戦略が、スピードよりも重要です。ゲーム終了画面でこの点が明確に示されます。
- **財務省が最も強力な戦略です。** ハミルトンにとって、公共信用と連邦財政は最も重要な手段でした。そのため、財務省が最も多く勝利します。ただし、商人や製造業者も十分に実行可能な戦略であり、ゲームの進め方も大きく異なります。

## ビルドと貢献

```bash
git clone https://github.com/mcp-tool-shop-org/sovereign
cd sovereign
npm install
npm test           # smoke tests
npm run verify     # full verify (smoke + pack-dry-run + CLI flag check)
npm run play       # open the game locally
```

リリースは、GitHub Actions (`release.yml`) を使用して、`v*` タグがプッシュされた際に npm に公開されます。Sigstore による信頼性の証明も行われます。ソースコードの信頼できるバージョンは、`main` ブランチにあります。
